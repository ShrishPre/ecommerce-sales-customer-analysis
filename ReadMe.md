# E-Commerce Sales & Customer Analytics Dashboard

##  About This Project

This is my capstone project where I worked on analyzing an e-commerce dataset (Amazon/Flipkart style) using SQL and Power BI.

Coming from a commerce background, I wanted to build something practical that connects business understanding with data analysis. This project helped me understand how sales, customers, and revenue are connected and how businesses use dashboards to take decisions.

---

## Objective

The main goal of this project was to:

- Analyze overall sales performance
- Understand customer buying behavior
- Identify repeat vs one-time customers
- Find top-performing cities and product categories
- Calculate important business KPIs

---

## Dataset Used

The dataset contains:

- Customers
- Orders
- Order Details
- Products
- Delivery Information
- Reviews

These tables were connected using proper relationships (One-to-Many model) inside Power BI.

## SQL Work

I used SQL to:

- Calculate total revenue
- Find customer lifetime value
- Identify repeat customers
- Analyze revenue by category
- Check cancelled and undelivered orders
- Perform aggregations using GROUP BY
- Use JOINs to connect multiple tables

This helped me understand how raw data is converted into meaningful business insights.

---

## Power BI Dashboard

I created three main report pages:

### Order Report
- Total Orders
- Cancelled Orders
- Late Deliveries
- Orders by City
- Orders by Category

### Sales Report
- Total Sales
- Total Quantity
- Average Order Value (AOV)
- Year-over-Year Growth
- Revenue by Category
- Monthly Sales Trend
- Top Products by Revenue

### Customer Report
- Total Customers
- Active Customers
- Repeat Customers
- Average Orders per Customer
- Customer Revenue
- Average Customer Spend
- Top Customers by Revenue
- Customers by City

I also added slicers for:
- Year
- Month
- Payment Mode
- Gender

This makes the dashboard interactive and more business-ready.

## Key Learnings

- A small group of customers contribute a large portion of revenue.
- Repeat customers are very important for long-term growth.
- Some product categories perform consistently better.
- Monitoring cancelled and late deliveries is important for customer satisfaction.
